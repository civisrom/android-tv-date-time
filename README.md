# Android TV Time Fixer

**AndroidTV Time Fixer** — это программа для управления настройками времени сервера NTP на устройствах Android TV через отладку ADB. Запускается в среде Windows как исполняемый файл .exe и работает через PowerShell. Программа позволяет установить сервер NTP по коду страны, назначить пользовательский сервер NTP, просмотреть текущие настройки устройства и получить информацию о подключенном Android TV.

## Основные возможности:
- **Изменение сервера NTP** по коду страны или установка пользовательского NTP-сервера.
- **Просмотр текущих настроек устройства** включая информацию о модели, версии Android, серийном номере, уровне заряда и текущем сервере NTP.
- **Генерация и использование ADB ключей** для безопасного подключения по TCP.
- **Подключение к устройству** с возможностью настройки повторных попыток и интервала задержки.

## Установка:
1. Скачайте .exe-файл из релизов.
2. Откройте PowerShell от имени администратора и перейдите в папку с файлом:

`cd "C:\путь\к\папке\с\программой"`

3. Запустите программу, введя команду:

`.\AndroidTVTimeFixer.exe`

## Использование:
1. Включите отладку ADB на Android TV:
2. Настройки → О устройстве → Сборка (нажмите 7 раз, чтобы активировать режим разработчика).
3. Зайдите в режим разработчика и активируйте отладку по сети
4. Запустите .exe файл, следуйте инструкциям, чтобы подключиться к устройству, установить NTP-сервер и посмотреть текущие настройки.

## Основные команды:
После запуска доступно меню:
- Изменение сервера NTP по коду страны: установите сервер по коду страны.
- Изменение NTP-сервера на пользовательский: настройте сервер вручную.
- Просмотр доступных кодов стран: список всех кодов стран и серверов NTP.
- Просмотр текущих настроек: информация о текущем устройстве и сервере NTP.
- Справка и выход.

## Диагностика
Программа выводит предупреждения и сообщения о возможных ошибках, таких как:
- Неверный формат IP-адреса.
- Проблемы с подключением к Android TV.
- Ошибки при попытке изменения NTP-сервера.

## Примечание:
Убедитесь, что Android TV и компьютер подключены к одной сети для успешного подключения.

## Поддерживаемые устройства
Программа предназначена для Android TV, таких устройств как Nvidia Shield, поддерживающих подключение через ADB по сети и управление временем через команду adb shell.
