# Android TV Time Fixer

**AndroidTV Time Fixer** — это программа для управления настройками времени сервера NTP на устройствах Android TV через отладку ADB. Запускается в среде Windows как исполняемый файл .exe и работает через PowerShell. Программа позволяет установить сервер NTP по коду страны, назначить пользовательский сервер NTP, просмотреть текущие настройки устройства.

## Основные возможности:
- **Изменение сервера NTP** по коду страны или установка пользовательского NTP-сервера.
- **Просмотр текущих настроек устройства** включая информацию о модели, версии Android, серийном номере, уровне заряда и текущем сервере NTP.
- **Генерация и использование ADB ключей** для безопасного подключения по TCP.

## Установка:
1. Скачайте AndroidTVTimeFixer-windows.zip файл из релизов, распакуйте его в папку.
2. Откройте PowerShell от имени администратора и перейдите в папку с файлом:

`cd "C:\путь\к\папке\с\программой"`

3. Запустите программу, введя команду:

`.\AndroidTVTimeFixer.exe`

## Использование:
1. Включите отладку ADB на вашем ТВ или Nvidia Shield
   Настройки > Настройки устройства > Об устройстве > Сборка (нажмите 7 раз или более)
2. Затем: Настройки устройства > Для разработчиков > Отладка по сети (Включить)
3. Установите время и дату в автоматический режим
   Настройки >  Настройки устройства > Дата и время > Автонастройка даты и времени > Использовать время сети
4. Ваш ТВ, Nvidia Shield и ПК должны быть подключены к одной сети
5. Определите какой ip адрес вашего устройства (ТВ или NVIDIA SHIELD), Настройки > Сеть и Интернет.

## Поддерживаемые устройства
Программа предназначена для Android TV, таких устройств как Nvidia Shield, поддерживающих подключение через ADB по сети и управление временем через команду adb shell.
