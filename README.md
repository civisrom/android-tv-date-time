AndroidTV Time Fixer — это программа для управления настройками времени и сервера NTP на устройствах Android TV через отладку ADB. Запускается в среде Windows как исполняемый файл .exe и работает через PowerShell. Программа позволяет установить сервер NTP по коду страны, назначить пользовательский сервер NTP, просмотреть текущие настройки устройства и получить информацию о подключенном Android TV.

Возможности
Изменение сервера NTP по коду страны или установка пользовательского NTP-сервера.
Просмотр текущих настроек устройства, включая информацию о модели, версии Android, серийном номере, уровне заряда и текущем сервере NTP.
Генерация и использование ADB ключей для безопасного подключения по TCP.
Подключение к устройству с возможностью настройки повторных попыток и интервала задержки.
Установка
Скачайте .exe-файл из релизов.
Запуск
Откройте PowerShell от имени администратора и перейдите в папку с файлом:
cd "C:\путь\к\папке\с\программой"
Запустите программу, введя команду: 
.\AndroidTVTimeFixer.exe
Использование
Включите отладку ADB на Android TV:
Настройки → О устройстве → Сборка (нажмите 7 раз, чтобы активировать режим разработчика).
Запустите .exe файл, следуйте инструкциям, чтобы подключиться к устройству, установить NTP-сервер и посмотреть текущие настройки.
Основные команды
После запуска доступно меню:

Изменение сервера NTP по коду страны: установите сервер по коду страны.
Изменение NTP-сервера на пользовательский: настройте сервер вручную.
Просмотр доступных кодов стран: список всех кодов стран и серверов NTP.
Просмотр текущих настроек: информация о текущем устройстве и сервере NTP.
Справка и выход.
Диагностика
Программа выводит предупреждения и сообщения о возможных ошибках, таких как:

Неверный формат IP-адреса.
Проблемы с подключением к Android TV.
Ошибки при попытке изменения NTP-сервера.
Примечание: Убедитесь, что Android TV и компьютер подключены к одной сети для успешного подключения.

Поддерживаемые устройства
Программа предназначена для Android TV, поддерживающих подключение через ADB по Wi-Fi и управление временем через команду adb shell.
