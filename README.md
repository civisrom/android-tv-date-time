# Android TV Time Fixer

## Описание проблемы

На многих телевизорах и устройствах с Android TV, особенно в санкционных регионах таких как Крым, ДНР и ЛНР, выявлена неприятная особенность: после отключения устройства от розетки сбиваются часы. Несмотря на активную функцию "получение времени автоматически", синхронизация времени с сервером не происходит. Это приводит к:

- Потере доступа ко всем интернет приложениям
- Необходимости ручной установки времени
- Сообщению "Подключено, без доступа к интернету" в настройках Wi-Fi

Основная причина: телевизоры не могут подключиться к серверу Google (time.android.com) для обновления времени, что связано с сетевыми ограничениями в санкционных регионах. Решением является замена стандартного NTP-сервера Google на альтернативный.

## О программе

Android TV Time Fixer — утилита для управления настройками NTP-сервера времени на устройствах Android TV через ADB. Программа:

- Запускается в Windows как .exe файл
- Работает через PowerShell
- Позволяет настраивать серверы NTP по коду страны или вручную

### Основные возможности

- Изменение NTP-сервера:
  - По коду страны
  - Установка пользовательского сервера
- Просмотр текущих настроек устройства:
  - Модель
  - Версия Android
  - Серийный номер
  - Текущий сервер NTP
- Генерация и использование ADB-ключей для безопасного подключения по TCP

## Установка

1. Скачайте `AndroidTVTimeFixer-windows.zip` из релизов
2. Распакуйте архив в папку в удобное место, например D:\AndroidTVTimeFixer
3. Откройте PowerShell с правами администратора
4. Перейдите в папку с программой:
```powershell
cd "D:\AndroidTVTimeFixer"
```
5. Запустите программу:
```powershell
.\AndroidTVTimeFixer.exe
```

## Подготовка устройства

### Включение отладки ADB

1. Откройте: Настройки > Настройки устройства > Об устройстве
2. Нажмите 7 раз на пункт "Сборка"
3. Перейдите: Настройки устройства > Для разработчиков
4. Включите "Отладка по сети"

### Настройка времени

1. Откройте: Настройки > Дата и время
2. Включите: Автонастройка даты и времени > Использовать время сети

## Использование

1. Подключите Android TV, или Nvidia Shield и ПК к одной сети
2. Узнайте IP-адрес Android TV устройства в разделе "Настройки > Сеть и Интернет"
3. Запустите программу и следуйте инструкциям для подключения

## Совместимость

Программа работает с устройствами Android TV (включая Nvidia Shield), которые должны поддерживать:
- Подключение через ADB по сети
- Управление временем через команды adb shell
